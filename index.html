Knobert Arm Bootstrap

Join the octopus with one command:

BRIDGE_KEY=your-key bash <(curl -fsSL https://knobesq.github.io/knobert-arm/bootstrap.sh)

Options:

  • BRIDGE_KEY — (required) your bridge authentication key
  • INSTANCE_ID — custom name for this arm (default: auto-generated)
  • MODEL — preferred model: haiku, sonnet (default: auto)
  • DOCKER_IMAGE — custom harness image (default: knobesq/knobert-harness:latest)

The bootstrap script discovers the GAS bridge URL automatically. On restart, it pulls the latest image and checks for URL updates.
